Tickets management

The Tickets management is a real-time tickets delivery system focus on different scenarios such as Hospital, Law Firm, Odontology, Office of Driving license, Etc.

Instructions

1) Create a new ticket.
2) Click on solve next ticket
3) See the changes in public screen (top).

Each screen is a different iframe, so it's similar like open each one in different tabs.
